くわしく

つかいかた（かんたん）

ホイールズーム（カーソル中心）
ドラッグ移動（パン）
クリック中心合わせ
ダブルクリック中心合わせ＋ズームイン
きれいに描く止め絵を高精細で仕上げ
Save PNG / S画像として保存
超深度超深度允许（ONなら深い領域で自動的にDeepNavに移行。HUDのactiveがONになります）
Reset / R初期位置へ
Tip: まずは適当にズームして、良い場所でHQ Render → Save PNG がおすすめです。
